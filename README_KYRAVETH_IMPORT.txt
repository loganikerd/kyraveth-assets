IMPORTANT:
- Keep each .gltf next to its matching .bin and referenced texture files.
- These models contain skins/armatures but the inspected glTF files contain no animation clips. Add locomotion animations separately (for example via Mixamo or a compatible free animation library).
- Use the Peasant outfit as the first in-game test because it better fits Kyraveth than the Superhero base clothing.
- Do not present these as final Kyraveth race designs. They are a professional-quality starter asset pipeline and can later be replaced or restyled.
